Google OAuth configuration
  1. Go to console.cloud.google.com and open your project.
  2. Enable the Google Drive API under APIs & Services → Library.
  3. Go to Credentials → your OAuth 2.0 Client ID → add your GitHub Pages URL to Authorized JavaScript origins and redirect URIs.
  4. On the OAuth consent screen, set publishing to Internal (Workspace only).
  5. Paste your Client ID below.
